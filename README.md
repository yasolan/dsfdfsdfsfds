### Что сдать

| `app.py` (или `main.py`) | Код Flask API |
| `README.txt` | Как запустить + список URL для Postman |

### Минимум запросов в Postman на защите (5 штук)

1. Успешный **GET** списка  
2. Успешный **POST** (создание)  
3. **GET** по `id` созданной записи  
4. Запрос с ошибкой **404** (несуществующий id)  
5. Запрос с ошибкой **400** (невалидные данные), если это есть в задании  


### Критерии практики (ориентир)

| Критерий | Баллы |
|----------|-------|
| Сервер запускается, API отвечает | 10 |
| GET (список / по id) | 15 |
| POST (создание, код 201) | 15 |
| Обработка ошибок (400, 404) | 15 |
| Корректный JSON в ответах | 10 |
| Читаемый код (маршруты, структура) | 15 |

**Данные:** хранить в памяти (список словарей в Python). База данных не обязательна.



## Шаблон кода (заготовка для практики)

```python
from flask import Flask, request, jsonify

app = Flask(__name__)

items = []
next_id = 1


@app.route("/items", methods=["GET"])
def get_all():
    return jsonify(items)


@app.route("/items/<int:item_id>", methods=["GET"])
def get_one(item_id):
    for item in items:
        if item["id"] == item_id:
            return jsonify(item)
    return jsonify({"error": "Not found"}), 404


@app.route("/items", methods=["POST"])
def create():
    data = request.get_json()
    if not data or "name" not in data:
        return jsonify({"error": "name is required"}), 400
    global next_id
    new_item = {"id": next_id, "name": data["name"]}
    next_id += 1
    items.append(new_item)
    return jsonify(new_item), 201


if __name__ == "__main__":
    app.run(debug=True, port=5000)
```

## Справочник: коды HTTP для экзамена

| Код | Когда использовать |
|-----|----------------------|
| 200 | Успешный GET, PUT, PATCH, DELETE |
| 201 | Успешное создание (POST) |
| 400 | Неверные данные в запросе |
| 404 | Ресурс не найден |
| 500 | Внутренняя ошибка сервера (по возможности избегать на экзамене) |
